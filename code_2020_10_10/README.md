# JmeterAuto 自动化测试项目

## 项目简介
这是一个自动化测试项目，包含接口自动化测试和性能测试相关代码。

## 项目结构
- `code_2020_10_10/` - 主要项目代码目录
  - `boot-order-service-consumer/` - 订单服务消费者
  - `boot-user-service-provider/` - 用户服务提供者
  - `gmall-interface/` - 通用接口模块
  - `path/` - 路径相关测试代码
- `apiAutoDemo/` - API 自动化测试示例

## 技术栈
- Java
- Spring Boot
- Jmeter
- Maven

## 使用说明
1. 克隆项目：`git clone https://github.com/daguideng/JmeterAuto.git`
2. 导入项目到 IDE
3. 根据需要运行测试

## 注意事项
- 项目包含 JDK 相关文件，体积较大
- 测试数据库脚本已清理
